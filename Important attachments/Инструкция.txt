ИНСТРУКЦИЯ к комплексу программ "Who Wants To Be A Millionaire? Control Program"

1. Распаковать всё содержимое архива в любую папку на вашем компьютере.
   Важно, чтобы все папки находились в одной директории с самой программой.
2. Запустить установку проигрывателя Winamp® из прилагаемого файла (папка "Installers").
   Проверьте, чтобы путь установки быд вида: "C:\Program Files\Winamp".
   При первом запуске проигрывателя нажмите "Сервис -> Уведомления -> Никогда не показывать".
   (последний пункт нужен только при установленной полной версии проигрывателя Winamp®)
3. Установите шрифты, прилагаемые к программе (Copperplate Gothic 30BC, ITC Conduit Cyrillic).
4. Запустите исполнительный файл программы (WWTBAM Control Program.exe).

*  При старте программы выберите нужную базу вопросов, требуемые для использования окна и их
   координаты расположения на экране, а также язык программы.

*  При использовании одного ПК для проведения игры, настройте второй экран компьютера на
   расширение изображения (желательно с разрешением 1024х768), оставьте активным только
   основное окно и окно зрителей (последнему задайте нужные координаты).

*  При использовании двух ПК, откройте программу на обоих устройствах (желательно в одном
   языковом режиме), потом в компьютере, который будет у режиссёра игры, впишите сетевой
   IP адрес второго ПК в поле в верхнем левом углу и нажмите соединение. Успешность соединения
   будет подтверждена на втором компьютере. Нажмите ОК. После этого компьютер можно не трогать.
	!Не забудьте отключить на втором ПК спящий режим и скринсейвер!

*  Во время игры следуйте указаниям подсветки кнопок:
	если сама игра не идёт (не задан вопрос), нажимайте кнопки алого цвета;
	если же игра началась, нажимайте жёлтые кнопки.

*  Двойной клик на вопросе из списка откроет вам редактор баз вопросов.
	если кликнуть на список в главном окне, откроется редактор стандартной базы вопросов;
	если кликнуть в отборочном туре, откроется редактор вопросов отборочного тура.

*  В окне отборочного тура в поля "Время" вводить только результаты ПРАВИЛЬНО ОТВЕТИВШИХ
   игроков.
	Разделителем десятичной дроби является (по умолчанию в системе) ЗАПЯТАЯ (",").
	После запятой должны стоять 2 знака.
	Числа должны быть меньше 20.
	Примеры: "16,45"; "8,01"; "9,30" и т.п.
	Если игрок не справился, результат - "0".
	Если победителей несколько, отборочный тур нужно переиграть, удалив из списка остальных
	   участников и выбрав другой вопрос.
	Кнопка "Таблица" используется дважды за один вопрос: первый для показа списка участников,
	   второй для демонстрации результатов отборочного тура.
